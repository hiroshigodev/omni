# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-03-15

### Added
- Specialized filters for **Docker build logs** and **SQL optimization**.
- **Wasm Edge Core** implementation (68KB binary).
- Unified reporting system via `omni-report.sh`.
- TTL support for LRU caching layer.
- Comprehensive English documentation (`ARCHITECTURE.md`, `DEVELOPMENT.md`).

## [0.1.0] - 2026-03-14

### Added
- Initial Zig core engine implementation.
- Basic Git and Build log filters.
- MCP Server gateway in TypeScript.
- Custom JSON-based rules for masking/removal.
